# Простой UDP сервер со своим протоколом
Позволяет обмениваться сообщениями между клиентами

# Установка
Заклоньте репозиторий:
```
git clone https://github.com/neyxezz/esp32-UDP-server
```
Затем перейдите в нее
```
cd esp32-UDP-server
```
Для удоства, в эту же директорию поставьте arduino-cli (или используйте arduino IDE для удобства)

Для моей платы (esp32 s3) я использую следующие команды для компиляции скетча и загрузки:
```
./arduino-cli compile -v --fqbn esp32:esp32:esp32s3 esp32.ino

./arduino-cli upload -p /dev/ttyUSB0 --fqbn esp32:esp32:esp32s3 .
```

Для подключения к серверу отредактируйте в файле `client.py` в последних строках `client = Client("192.168.0.212", 8888)` на ваш айпи адрес, полученный в мониторе порта, в который можно перейтив через
```
./arduino-cli monitor -p /dev/ttyUSB0 --config baudrate=115200
```

# Особенности
* Пока устройство не подключено к WiFi, оно будет мигать красным цветом, при подключении - статичным зеленым
